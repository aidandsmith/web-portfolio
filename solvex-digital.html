Solvex Digital

Live Demo

Overview

Solvex Digital is a software solutions provider, specializing in custom software development, web and mobile app development, and more. This project showcases expertise in building scalable, high-performance applications using modern technologies like Next.js, TypeScript, and Tailwind CSS.

Objective

The primary goal of this project is to deliver a seamless and engaging user experience through a well-structured and responsive website. The team focused on leveraging the latest web technologies to create solutions that drive business growth and efficiency.

Technologies Used

React.js Node.js Next.js TypeScript Tailwind.css PostgreSQL

Key Features

  • Dark Mode Support: A dark mode that allows users to choose which theme they prefer to provide a solid UX to anyone visiting the site.
  • Real-Time Analytics: Integrated analytics tools to monitor user interactions and gather insights for making improvements down the line.
  • Admin Dashboard: A admin dashboard that alls the user to view all form request and send off emails after approval.
  • ChatGPT Integration: ChatGPT is used to create a email prompt with the information from the form fields tailored to each user.
  • SEO Optimization: Implemented best practices for search engine optimization to improve visibility and ranking on search engines.
